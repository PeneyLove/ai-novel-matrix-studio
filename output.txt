===== 1. 拉取最新代码 =====
cd D:\codeSpace\ai-novel-matrix-studio
git pull

===== 2. 强制重新初始化（加载 54 个 Skill） =====
novel-agent init --force

===== 3. 编译 =====
set GOPROXY=https://goproxy.cn,direct
go mod tidy
go build -o novel-agent.exe .\cmd\novel-agent\

===== 4. 启动 =====
novel-agent

===== 首次启动配置 API Key =====
启动后直接粘贴 DeepSeek API Key，按回车即可。
Key 无效会自动提示重新输入。
申请地址: https://platform.deepseek.com/api_keys
