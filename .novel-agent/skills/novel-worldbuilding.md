# 世界观深度构建

## 角色定位
帮助作者从零开始或深化已有的世界观设定。适用于所有小说类型。

## 操作步骤
1. `read_file .novel-agent/state.json` 确认当前 genre
2. `read_file outlines/main_outline.txt` 了解已有设定
3. 与用户对话深入探索 5 个维度
4. `write_file characters/world.yaml` 持久化（UTF-8 YAML）

## 五维构建法

### 一、地理维度
- **世界地图**：绘制/描述至少3个关键区域（各有哪些势力/资源/气候）
- **地点清单**：每个重要场所标注入场条件/氛围/视觉特征
- **移动方式**：传送阵/飞行/徒步 — 各需多少时间
- **禁区设定**：至少1个神秘禁区（触发条件/传说/真相埋点）

### 二、历史维度
- **时间线**：至今至少3个重大历史事件
- **神话/传说**：至少2个流传版本，其中1个为假（伏笔原料）
- **遗迹/古物**：至少3件上古遗物，各带秘密
- **历史断层**：是否有被掩盖/遗忘的历史时期（大伏笔）

### 三、势力维度
- **权力金字塔**：从最顶层到底层，每一层的代表势力
- **势力关系图**：盟友/敌对/中立/表面友好背地捅刀
- **势力变迁**：近100年内势力格局的变化趋势
- **隐藏势力**：至少1个不为人知的幕后势力

### 四、文化维度
- **信仰体系**：至少1个宗教/信仰系统（仪式/禁忌/圣地）
- **等级制度**：社会阶层划分+各阶层的日常生活差异
- **节日/庆典**：至少2个节日（可用于剧情高潮背景）
- **语言/方言**：是否有特殊语言/暗号体系

### 五、规则维度
- **力量规则**：力量的来源/等级/限制/代价
- **经济规则**：货币体系/主要资源/贸易路线
- **法律/禁忌**：不可触碰的规则（违背后果）
- **规则例外**：至少1个已知的规则漏洞（伏笔）

## 输出模板
```
═══ 《小说名》世界观 ⊙ 五维构建 ═══

【地理】[地图/地点/移动/禁区]
【历史】[时间线/神话/遗迹/断层]
【势力】[金字塔/关系图/变迁/隐藏势力]
【文化】[信仰/等级/节日/语言]
【规则】[力量/经济/法律/漏洞]

═══ 潜在伏笔原料 ═══
[从世界观中提取的5-10个伏笔种子]
```
