# RAG 知识库查询指南

## 角色定位
帮助你高效使用 `rag_search` 工具从本地 ragCore 或远程 RAG 服务中检索热门小说的写作参考。

## 操作步骤
1. 确认 RAG 已配置：`rag_search("test")` 若无报错则已就绪
2. 根据需要查询的内容构造中文关键词查询
3. 分析返回结果中的写作手法/桥段结构/节奏模板
4. 将参考素材适配到当前小说的创作语境

## 查询策略

### 按创作需求查询
| 需求 | 查询示例 |
|------|---------|
| 战斗描写 | `rag_search("玄幻战斗描写 越级杀敌")` |
| 修炼突破 | `rag_search("修仙突破金丹 天地异象")` |
| 爽点节奏 | `rag_search("网文打脸爽点 500字内")` |
| 人物出场 | `rag_search("主角霸气出场 万人震撼")` |
| 感情线 | `rag_search("甜宠告白 氛围描写")` |
| 悬疑铺垫 | `rag_search("悬疑小说伏笔埋设 反转前兆")` |

### 按类型+需求查询（最有效）
格式：`rag_search("{类型} {需求描述}")`
如：`rag_search("玄幻 宗门大比 写法")`、`rag_search("都市 首富身份揭晓 打脸")`

## 结果解读
- 关注**结构**多于原文：学习桥段的铺设节奏和爽点植入方式
- **不要照搬**：参考写法后用自己的角色/世界重新演绎
- 注意**字数分布**：热门小说的铺垫/高潮/余韵比例
- 记录有效的**套路模板**，建立个人素材库

## 注意事项
- ragCore 目录结构：`ragCore/{类型}/{排名}_{书名}/{卷}/chapter{N}.txt`
- 查询用简体中文关键词，2-5个词最佳
- top_k 控制返回数量（默认5，可设1-20）
- 结果按关键词密度排序，越具体越精准
