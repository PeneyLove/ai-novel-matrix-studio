# 都市网文 — 节奏优化

## 操作步骤
1. `read_file` 目标章节（含前后各2章）
2. 分析节奏曲线→标记拖沓/过快
3. `write_file` 覆写

## 诊断标准
- **节奏波动**：连3章无高潮 → 过慢；连2章高潮 → 过赶
- **段落推进**：单段>500字无实质推进 → 压缩
- **过渡章节**：超半章纯过渡（赶路/修炼日常） → 精简或加微爽点
- **断章点**：每章结尾必有钩子

## 修复策略
- 拖沓：删描述/信息移到对话/合并短场景
- 过赶：加内心独白/环境/配角反应缓冲
- 玄幻节奏模板：日常→冲突→战斗→突破→余韵（每卷循环）
