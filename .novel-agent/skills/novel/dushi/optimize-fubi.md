# 都市网文 — 伏笔回收

## 操作步骤
1. `read_file .novel-agent/hooks/ledger.yaml` 读取台账
2. `read_file` 读取目标章节
3. 核对 pending 伏笔→标记临期→输出回收建议
4. 用户确认后 `write_file` 覆写 + 更新台账

## 诊断标准
- **临期伏笔**：超预定回收章3章以上 → 警告
- **密度**：回收1个必须新埋1-2个
- **回收质量**：禁止巧合回收（偶遇/恰好/正好），必须有伏笔→触发→揭示→反转链条
- **反转强度**：每次回收≥1个反转

## 修复策略
- 设计回收剧情：铺垫300字→触发200字→揭示300字→反转200字→余波
- 延误伏笔：紧急回收（压缩到下章）/ 升级延迟（小伏笔升为中伏笔延长回收期）
