# 都市网文 — 正文定向续写（大纲绑定版）

## 前置条件
- `outlines/main_outline.txt` / `.novel-agent/hooks/ledger.yaml` / `.novel-agent/state.json`

## 操作步骤
同玄幻修仙：读进度→读大纲→读台账→读上文→撰写→持久化

## 都市网文续写规范

### 章节结构（每章2000-3500字）
1. **钩子回收**（100-200字）：衔接上章
2. **本章正文**（1500-3000字）：都市生活/冲突/博弈
3. **微爽点植入**：每章≥1处（打脸/实力展示/身份压制/财富展示/情感升温）
4. **伏笔铺垫**：每章1处
5. **收尾钩子**（50-100字）

### 写作风格
- **对话占比40-50%**：都市文对话驱动，但每人说话风格不同
- **场景真实**：使用真实地名/品牌替代（如"江城""帝豪酒店""劳斯莱斯"）
- **身份反差**：通过他人态度变化制造爽感
- **商战有逻辑**：商业博弈要有专业感（收购/对赌/股价/合同）
- **武力有分寸**：都市战神/异能，战斗要有克制（法律/隐藏身份限制）
- **禁止AI套话**

### 常用元素模板
- **打脸描写**：轻视→暗讽→当众碾压→身份揭晓→震惊四座
- **身份揭晓**：匿名参加→被讥讽→关键人物行礼→"见过XX大人"
- **商战描写**：对手出招→主角预判→反杀→对手崩溃→收获

```markdown
- 植入爽点：[类型]
- 埋伏笔/回收：hook-XXX

[正文...]

---

*爽点/伏笔/钩子摘要*
```
