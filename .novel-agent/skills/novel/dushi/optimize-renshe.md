# 都市网文 — 人设优化

## 操作步骤
1. `read_file` 近10章 + `read_file characters/protagonist.txt`
2. 检查行为一致性/记忆点/配角功能
3. 输出诊断 → 用户确认 → `write_file` 覆写 + 更新 characters/

## 诊断标准
- **行为一致**：是否OOC？动机是否自洽？
- **记忆点**：主角有标志性特征/口头禅/功法特点？
- **配角防工具人**：每个配角独立动机？非纯推动剧情？
- **反派智商**：手段合理？动机充分？压迫感够？
- **弧光**：10章内角色有可感知变化？

## 修复策略
- OOC → 回顾人设→改写行为→内心戏解释
- 工具人 → 加独立对话/动机场景
- 扁平反派 → 加智商展示/策略博弈
