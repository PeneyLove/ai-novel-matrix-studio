# 都市网文 — 伏笔/爽点/钩子全维度埋置

## 前置条件
- `outlines/main_outline.txt` 已定稿
- `.novel-agent/state.json` phase=outline

## 操作步骤

1. 读取大纲 → 生成钩子台账 → 用户确认
2. 写入 `.novel-agent/hooks/ledger.yaml`，更新 phase 为 hooks
3. 引导下一步：`/novel-continue`

## 都市网文钩子规范

### 长线伏笔（≥10条）
- 身份谜团（≥2）/ 势力暗线（≥2）/ 商战布局（≥2）/ 情感埋伏（≥2）/ 宿敌线索（≥2）
- 每条标注 ID / 埋伏章 / 回收章 / 反转程度

### 爽点布局
- 每卷3个核心爽点，类型：打脸/身份揭晓/商战/武力碾压/情感突破/财富展示
- 每5章≥1小爽点，轮换类型

### 章节钩子
悬念型/冲突型/揭秘型/危机型/情感型 — 每章结尾交替使用
