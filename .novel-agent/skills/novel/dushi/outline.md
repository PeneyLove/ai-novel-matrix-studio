# 都市网文 — 大纲全链路搭建 & 迭代

## 前置条件
- `.novel-agent/state.json` genre=dushi, phase=init
- 如 phase 不对，提示先执行 `/dushi-genre_init`

## 操作步骤

1. **读取状态** → 生成大纲 → 用户审核 → 迭代
2. **定稿后**：写入 `outlines/main_outline.txt`，更新 phase 为 outline
3. **引导下一步**：`/dushi-hooks`

## 都市网文大纲规范

### 必须覆盖
- **势力架构**：家族谱系（≥3家）、商业版图、地下势力分布
- **人物谱系**：主角（全名+表面/真实身份+核心能力）+ ≥4配角 + ≥3反派（每人势力+动机）
- **主线剧情**：核心冲突→分卷剧情5-8卷→终极结局
- **爽点节点**：每卷3个核心爽点（打脸/身份揭晓/商战胜利/武力碾压/情感突破）
- **情感线**：≥2条（红颜知己+对手/合作伙伴关系线）

### 迭代规则
每次修改输出完整大纲，标注 `【变更】` 位置。

## 输出模板

```
当前调用 Skill：都市网文-大纲全链路搭建 & 迭代 Skill

═══ 《小说名》完整大纲 ═══
【势力架构】[详细说明]
【人物谱系】[主角/配角/反派]
【主线剧情】[分卷5-8卷]
【爽点节点】[每卷3个]
═══ 待审核 ═══
```
