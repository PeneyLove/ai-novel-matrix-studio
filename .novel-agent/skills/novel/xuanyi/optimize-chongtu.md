# 悬疑灵异 — 冲突升级

## 操作步骤
1. `read_file` 近10章 + `read_file outlines/main_outline.txt`
2. 梳理所有冲突线→分析升级趋势→标记停滞→设计催化剂
3. `write_file` 覆写

## 诊断标准
- **冲突线数量**：主线/支线/人际/内心 — 至少4线并行
- **升级趋势**：每条线10章内有升级？
- **停滞冲突**：同类冲突重复无质变 → 需要催化剂
- **新维度**：引入新冲突源的空间？

## 修复策略
- 停滞线 → 催化剂事件（新人物/新信息/新威胁）
- 平行升级 → 主线+支线+人际同时推
- 冲突交织 → 主线影响人际→人际影响支线
- 玄幻常见冲突源：诡异规则/幕后真凶/角色秘密/时间压力/信任崩塌
