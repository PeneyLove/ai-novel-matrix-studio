# 悬疑灵异 — 类型定型 & 初始化
## 操作步骤
1. `read_file .novel-agent/state.json`
2. 确认细分：灵异探险 / 规则怪谈 / 刑侦悬疑 / 校园诡异 / 民间怪谈 / 无限悬疑
3. 确认核心套路：层层揭秘 / 规则破解 / 真相反转 / 生存博弈 / 因果报应
4. `write_file .novel-agent/state.json`
5. 引导 `/xuanyi-outline`

## 创作规范
- **规则体系**：悬疑/灵异必须有清晰规则（鬼怪限制/怪谈触发条件/副本机制）
- **伏笔密度**：悬疑类型伏笔密度×2，每章至少2处线索
- **氛围优先**：环境描写占比>25%（光线/声音/温度/气味）
- **反转节奏**：每卷1个大反转，每10章1个小反转
- **字数**：每章2500-4000字
