# 古言权谋 — 伏笔/爽点/钩子全维度埋置

## 操作步骤
读大纲→生成台账→用户确认→写入 `.novel-agent/hooks/ledger.yaml`→`/novel-continue`

## 钩子规范
- **长线伏笔（≥10条）**：身世谜团/朝堂暗线/后宫布局/边疆危机/江湖势力
- **爽点**：每卷3个核心爽点（扳倒对手/筹谋成功/身份反转/圣眷/兵权），每5章≥1小爽点
- **章节钩子**：悬念/冲突/揭秘/危机/情感 — 交替使用
