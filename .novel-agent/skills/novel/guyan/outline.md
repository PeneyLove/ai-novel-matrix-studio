# 古言权谋 — 大纲全链路搭建 & 迭代

## 操作步骤
读状态→生成大纲→迭代→定稿后写入 `outlines/main_outline.txt`，更新 phase→`/guyan-hooks`

## 大纲规范
- **势力架构**：后宫格局+前朝势力+世家谱系+边疆军事
- **人物谱系**：主角（身份+前世/现代记忆+目标）+ ≥4配角（盟友/敌人/摇摆者）+ ≥3反派
- **主线**：核心冲突（夺嫡/复仇/保家/上位）→分卷5-8卷→结局
- **爽点**：每卷3个（扳倒对手/身份揭晓/筹谋成功/皇帝青睐/战场胜利）
- **情感线**：≥1条（帝王/王爷/将军/谋士）

## 输出模板
```
═══ 《小说名》完整大纲 ═══
【朝代设定/势力架构/人物谱系/主线/爽点】
═══ 待审核 ═══
```
