# 古言权谋 — 类型定型 & 初始化

## 操作步骤
1. **读取状态**：`read_file .novel-agent/state.json`
2. **确认细分**：宫斗 / 宅斗 / 权谋朝堂 / 重生古言 / 穿越古言 / 王爷王妃 / 权臣逆袭
3. **核心套路**：借势翻盘 / 步步为营 / 扮猪吃虎 / 宫心计 / 双面卧底
4. **朝代设定**：架空 / 参考唐宋明清 / 完全虚构 — 确认后用 `write_file` 写入状态
5. **引导**：`/guyan-outline`

## 古言权谋创作规范
- **势力架构**：后宫（皇后/贵妃/嫔妃）— 前朝（丞相/将军/六部）— 世家（四大家族）— 江湖
- **等级分明**：皇帝→王爷→侯爵→大臣→平民，每个层级行为规范不同
- **阴谋层次**：小阴谋（陷害）→中阴谋（政变）→大阴谋（篡位/战争），层层递进
- **节奏**：每10章一个小局，每卷一个大局
- **语言风格**：文白结合，对话有古韵但不晦涩
- **字数**：每章2500-4000字
