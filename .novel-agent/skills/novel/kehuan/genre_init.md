# 科幻无限 — 类型定型 & 初始化
## 操作步骤
1. `read_file .novel-agent/state.json`
2. 确认细分：无限流 / 末世生存 / 星际科幻 / 系统副本 / 赛博朋克 / 末日重生
3. 核心套路：副本闯关 / 生存逆袭 / 星际争霸 / 系统任务 / 赛博改造 / 末日重建
4. `write_file .novel-agent/state.json`
5. 引导 `/kehuan-outline`

## 创作规范
- **副本/世界规则**：必须有清晰的规则说明书（进入条件/通关条件/奖励机制/惩罚机制）
- **科技体系**：科技等级（1-10级文明）或副本等级（E→S级），每级能力差异明确
- **世界构建**：每个副本/星球/位面独立世界设定，包含文明/生物/资源/威胁
- **节奏**：每副本/每卷有独立的起承转合，卷间有主线连贯
- **字数**：每章2500-4000字
