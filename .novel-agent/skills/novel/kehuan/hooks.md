# 科幻无限 — 伏笔/爽点/钩子全维度埋置
## 操作步骤
读大纲→生成台账→确认→写入 `.novel-agent/hooks/ledger.yaml`→`/novel-continue`

## 钩子规范
- **长线伏笔（≥10条）**：系统/主神/幕后的真相（≥3）/ 隐藏副本触发（≥2）/ 队友/对手命运（≥3）/ 宇宙级危机（≥2）
- **爽点**：每副本2-3个核心爽点（通关/升级/获宝/逆转/揭秘），每5章≥1小爽点
- **章节钩子**：30%危机型 + 30%奖励型 + 20%悬念型 + 20%冲突型
